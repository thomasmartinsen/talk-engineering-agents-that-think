# using qdrant as memory connector in agents

## Qdrant
Download and run qdrant:
- docker pull qdrant/qdrant
- docker run -p 6333:6333 -p 6334:6334

Local qdrant dashboard:  
- http://localhost:6333/dashboard#/collections
